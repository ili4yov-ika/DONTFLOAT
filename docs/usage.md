# Использование DONTFLOAT

## Установка (Linux/macOS)

```bash
git clone https://github.com/yourname/dontfloat.git
cd dontfloat
./scripts/build.sh
./scripts/package.sh

Загрузка аудио
Откройте приложение.

Меню File → Open Audio File...

Выберите аудиофайл (.wav, .mp3, .flac и др.)

Волна появится на экране.

Анализ BPM
После загрузки BPM автоматически анализируется.

Если темп нестабилен, отобразится:

~124 BPM — приблизительное значение

Всплывающее окно предложит выровнять темп

Выравнивание BPM
Выберите Edit → Align BPM (будет доступно после анализа).

Результат отображается поверх исходной волны (или создаётся новая дорожка).

В будущем планируется экспорт выровненного файла.

Горячие клавиши
Команда	Сочетание
Открыть файл	Ctrl + O
Открыть последний	Ctrl + L
Выровнять BPM	Ctrl + B
Выход	Ctrl + Q

Сохранение
Меню File → Save As...

Поддерживаются форматы .wav, .flac

Настройки
DONTFLOAT сохраняет:

последний открытый путь

предпочтительный формат экспорта (будущее)

зум-уровень отображения

Настройки хранятся в QSettings (на платформе по умолчанию).

Поддержка
