# DONTFLOAT Tests/source4test

Этот каталог содержит .mp3 файлы для отладки механизмов программы

## Структура

- `*_C120BPM.mp3` - Музыкальный файл с постоянными 120 BPM
- `*_V120BPM.mp3` - Музыкальный файл с переменными 120 BPM
- `*AA_C*BPM.mp3` - Музыкальный файл AA постоянными BPM
- `*AA_V*BPM.mp3` - Музыкальный файл AA переменными BPM
